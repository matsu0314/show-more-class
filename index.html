もっと見るボタンをJavaScriptで実装

class="showmore_wrap"を設定した要素にもっと見るボタンを追加します。
以下は最も基本的な使い方です。 

window.addEventListener('load', () => {
    const showmore = new ShowMore();
  });

初期値は次の通りです。

window.addEventListener('load', () => {
  const showmore = new ShowMore({
      selector: '.showmore_wrap', //セレクタ
      prefix = "shm", // CSSのプレフィックス
      displayShowHeightPc: 0.3, //最初に表示する要素の高さ（PC）
      displayShowHeightSp: 0.2, //最初に表示する要素の高さ（SP）
      closeBtnPositionTop:true, // 閉じるボタンを上に表示するか
      headerHeightPc: 0, //ヘッダー分高さを調整（PC）
      headerHeightSp: 0, //ヘッダー分高さを調整（SP）
      showpc: true, // PCでもっと見るを実装
      showsp: true,// SPでもっと見るを実装
    });
  });
